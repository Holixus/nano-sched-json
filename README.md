# nano-sched-json
JSON parsing and stringifying nano-sched plugin

## data object

* encoding `<String>`
* content `<any>`


## Functions


### json.parse

Converts `{ encoding:'utf8', content: '...' }` into `{ encoding:'json', content: Object }`.


### json.stringify

Converts `{ encoding:'json', content: Object }` into `{ encoding:'utf8', content: '...' }`.
